A **phishing email** is a fake message designed to trick you into:

- Clicking a dangerous link  
- Giving personal information (resume, phone number, login credentials, bank details)  
- Downloading harmful attachments  
- Believing false claims (job offer, warnings, urgent alerts)

Attackers make the email look professional so users trust it.

Phishing = **Deception + Social Engineering**
